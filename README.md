# Aura Hydration - Sustainable Luxury Landing Page

A premium, minimalist landing page for a sustainable water bottle brand.

## Features
- **Art Direction**: Sustainable Luxury aesthetic with Playfair Display and Inter.
- **Animations**: Smooth scroll and entry animations using Framer Motion.
- **Responsive**: Fully optimized for mobile and desktop.
- **Components**: Built with React, Tailwind CSS v4, and Shadcn UI.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev:client
   ```

3. **Open the application**:
   Navigate to [http://localhost:5000](http://localhost:5000) in your browser.

## Tech Stack
- React 19
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide Icons
- Wouter (Routing)
